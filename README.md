## ⚡ Executive summary (одностраничник)
- **Что продаём клиенту:** не «сайт», а **фазовое цифровое партнёрство** для мирового бренда. Он пришёл за сайтом — даём сайт + видение роста.
- **Ниша (наш козырь):** эталоны (Blue Safari/Alphonse/FlyCastaway) продают **по его же модели** (пакет → «от $» → лид к менеджеру), но **EN-only**. На Мальдивах **в открытой выдаче нет fishing-first премиум-бренда EN+RU**; свободен конверсионный кластер **«рыбалка + отель + менеджер одним пакетом»**. Низкую осведомлённость рынка («90% не знают») — качественно подтверждаем.
- **Модель сайта:** лид→менеджер (не букинг/оплата), EN-first мультиязык (**старт EN + DE + RU**), fishing-first, визуализированный дефицит, контент/видео, **CRM + WhatsApp с Фазы 1**.
- **Скрытый актив:** YouTube-канал клиента → потенциал для нового AI-поиска (структуру надо перепаковать — это наша работа).
- **Деньги (на созвоне дать вилку «в комплексе»):** A $4–8k · **B $12–25k (реком.)** · C $30–50k+ · айдентика +$3–10k · ретейнер $2.5–7k/мес. Окупаемость считать **через маржу клиента**, не через цену тура.
- **Ход закрытия:** платный **Discovery $1.5–4k** (зачёт в проект) — де-риск точной сметы, а не «плата за прайс». Затем Фаза 1 → ретейнер.
- **Следующий шаг:** провести созвон по [playbook (19)](docs/19-next-call-playbook.md) → закрыть на Discovery.
