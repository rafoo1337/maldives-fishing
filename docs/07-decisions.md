# 07 — Решения (лог)

## ✅ Принято
- Сайт — инструмент продажи (**лид → менеджер**), НЕ онлайн-букинг/оплата.
- Личный кабинет пользователю — **нет**. CRM — **да** (копить лиды, не терять заявки).
- **EN-first**, мультиязык; посадки под рынки.
- **Fishing-first:** рыбалка — герой, отель — крючок.
- 3 вида рыбалки (Fly / Big game / Sport) + обучалки + безопасность + FAQ.
- 2 контактных номера (MV + RU) + WhatsApp.
- Делаем под **Модель 1**, проектируем так, чтобы не мешать **Модели 2**.
