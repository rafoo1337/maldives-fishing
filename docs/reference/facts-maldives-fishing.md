## 1. Виды и трофеи по типам рыбалки

### Fly fishing (флэтс) — что РЕАЛЬНО есть vs Сейшелы
**[важно для бренда]** Не ставить Мальдивы в один ряд с Сейшелами как «бонфиш-дестинацию» — это неточно.
- **Уверенно продаём:** **GT (giant trevally)** — главный флэтс/риф-трофей, круглый год, мировой уровень; прочие каранксы (bluefin, golden, yellow-spotted); **триггерфиш** (yellowmargin, titan) — в изобилии.
- **Бонфиш** — есть на внешних флэтс, но **в разы меньше, чем на Сейшелах**; подавать как «бонус к люкс-отдыху, а не главный объект». На Сент-Брэндоне/Альфонсе бонфиш мигрирует косяками — там эталон, мы не конкурируем.
- **Permit** `[спорно]` — заявляется гидами, но без подтверждённой численности/вида для Мальдив → «редкий bucket-list прилов», без обещаний.
- **Milkfish** `[оценка]` — нишевый, эпизодический трофей для продвинутых.
> Формула: «GT и триггерфиш — ядро; бонфиш — бонус; пермит/milkfish — редкие трофеи».

### Big game / троллинг / глубоководка
Выход за атолл, троллинг по «признакам жизни». Объекты: **yellowfin tuna**, **dogtooth tuna**, **wahoo**, **mahi-mahi/dorado**, **sailfish**; **marlin** — нечастый/эпизодический (писать «преимущественно голубой, возможный, но не гарантированный» `[спорно]`); skipjack — массовый.

### Sport / джиггинг-попперинг
Спокойная вода → джиг/поппинг/троллинг «лёгкие в исполнении». **GT** (поппинг над рифами, ~14–36 кг `[оценка одного источника]`), **dogtooth tuna** (джиггинг у свалов), snapper/grouper/jobfish/emperor, wahoo, yellowfin.
- **Dogtooth tuna:** вид до ~130 кг; **рекорд IGFA 107.5 кг — Танзания, НЕ Мальдивы** `[важно]`. Корректно: «Мальдивы — одно из лучших мест мира по собачьему тунцу», без присвоения рекорда.

## 2. Сезоны и муссоны
| Муссон | Название | Месяцы | Характер |
|---|---|---|---|
| Северо-восточный | **iruvai** | ~дек–апр | Сухо, ясно, спокойное море `[спорно по границам месяцев]` |
| Юго-западный | **hulhangu** | ~май–ноя | Влажно, ветренее, апвеллинги (богатая кормовая база) |

- **Big game:** пик в сухой сезон, **ноя–мар** (часть источников до апр), ярче всего янв–мар.
- **Круглый год:** GT, собачий тунец, полосатый тунец.
- **Низкий сезон по погоде ≠ плохая рыбалка** (апвеллинги стягивают пелагиков).
- **Nakaiy** — традиционный мальдивский календарь ~13–14-дн. периодов, до сих пор используется рыбаками → сильный аутентичный нарратив.
- **Закрытые атоллы = штиль** `[оценка]`: барьерный риф = волнолом, лагуна укрыта; в любой муссон одна сторона атолла подветренная → ловим почти при любой погоде. Подавать как аргумент бренда.

## 3. География
- **Атолл** — кольцо рифов вокруг лагуны. **Слово «атолл» — от мальдивского *atholhu*** → сильный факт для сторителлинга.
- Мальдивы = **26 природных атоллов**, ~820–960 км `[спорно — разброс источников]`.
- **Хувадху (Huvadhoo/Gaafu):** ~2 900 км² `[спорно — источники дают 2900–3150]`, **255 островов**, **рекорд Гиннесса — больше всего островов в мире** `[важно — железобетонный факт, на него и опираться]`; один из крупнейших атоллов мира (ранг по площади НЕ присваивать — спорно). Глубокий канал Huvadhu Kandu (One-and-a-Half-Degree Channel) → сильные течения, пелагики (аргумент для big game). Габариты лагуны 82×65 км, до ~85 м `[спорно — тур-источник]`.
- **Логистика:** всё на одном острове-курорте, рифы/флэтс/свалы в минутах хода (vs Сейшелы — лучшие флэтс на удалённых внешних атоллах с отдельным перелётом). Аргумент «рыбалка без жертв со стороны семьи».

## 4. Регуляторика (фундамент «премиум = безопасность + экология»)
- **Охраняемые виды:** **акулы** — общенациональный запрет с 15.03.2010 (Мальдивы 2-я страна после Палау, объявившая EEZ акульим заповедником). ⚠️ **Запрет позже частично смягчён** (напр. лов gulper shark разрешён ~дек–июнь) → в present-tense «действующий акулий заповедник» НЕ утверждать, статус сверять на дату публикации `[важно]`. Также: **наполеон/humphead wrasse** (с 1995); **китовая акула**, **дельфины/киты** (с 1993); **черепахи** (запрет ~1995 `[спорно по дате — сверить EPA]`); манты/скаты `[спорно по охвату]`; гигантские моллюски, чёрный коралл, лобстеры (самки с икрой / <25 см).
- **Заповедные зоны (MPA):** 15 дайв-сайтов с 1996, +10 с 1999, далее росло; в них лов запрещён.
- **Catch-and-release:** GT и билфиш — преимущественно C&R. Норма: «не ешь — отпусти».
- **Лицензии:** для гостя на чартере отдельная лицензия обычно НЕ нужна (покрыта разрешением оператора); shore fishing с пляжа резорта — часто запрещён без разрешения менеджмента. Лов охраняемых видов — крупные штрафы.
> Нарратив: catch-photo-release, барблесс-крючки, ноль давления на охраняемые виды — этика + соответствие закону + статус.

## 5. Почему Мальдивы лучше Сейшел для «рыбалка + семья»
Подавать **только в связке с рыбалкой** (в целом по «семейности» Сейшелы тоже сильны):
1. **Логистика:** ловишь полдня и вернулся к семье; на Сейшелах топ-рыбалка = отдельная экспедиция на удалённые атоллы.
2. **Спокойная вода атоллов** (комфорт и для нерыбачащих, и для флэтс).
3. **Год-раунд рыбалка + курорт.**
4. **Contained island** — меньше стресса родителям (детские клубы, мелкие лагуны).
5. **Где Мальдивы реально сильнее:** GT (круглый год, мировой уровень), собачий/жёлтопёрый тунец, риф-джиггинг. **Честно:** по бонфишу/пермиту Сейшелы превосходят — не скрываем, перенаправляем на сильные стороны.
> Формула: «Сейшелы — для фанатика бонфиша, готового к экспедиции. Мальдивы — для тех, кому нужны GT/тунец мирового класса + семья, люкс и спокойная лагуна в шаге от рыбалки».
