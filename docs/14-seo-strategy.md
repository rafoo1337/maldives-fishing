# 14 — SEO / GEO-стратегия (по регионам)

> Собрано агентами по методологии **claude-seo**: SERP-overlap семантическая кластеризация + GEO (AI-поиск). Частотности — качественные (точные нужно снимать Wordstat/Keyword Planner с гео).

## TL;DR
1. **Уникальный SERP-угол:** агрегаторы продают ЛИБО почасовой чартер, ЛИБО отель. Никто не упаковывает **«трофейная рыбалка + премиум-отель + персональный менеджер» одним пакетом** → главный конверсионный кластер (`fishing package/resort/holiday maldives`) почти свободен.
2. **Не бить во фронт по `fishing maldives`** (захвачен FishingBooker) — заходить через свободный коммерческий wedge + long-tail видов/атоллов.
3. **Языки на старте: EN + DE + RU** (обоснование ниже). Остальное — per-market в ретейнере.
4. **GEO/AI-поиск — недооценённый рычаг:** у клиента уже есть **YouTube-канал** (корреляция с AI-цитированием ~0.737 — сильнейший сигнал). Это актив, которого нет у конкурентов.
5. **Технический моат:** корректный `hreflang` + быстрый мобайл + schema — то, чего нет у Boombastic и локалов.

## Часть 1. EN — ключевые кластеры (hub-and-spoke)

Четыре pillar-страницы. Pillar 1 — трафик-магнит (инфо), Pillar 2/4 — деньги, Pillar 3 — нишевый авторитет.

**PILLAR 1 — `/fishing-in-maldives`** («Fishing in the Maldives: The Premium Angler's Guide») — перехват инфо-трафика у FishingBooker.
Spokes (кластер «Plan & Know»): `best time/season`, `best atolls / north male / ari atoll`, `fishing license & rules` (E-E-A-T), `reef/bottom/night fishing`, `fishing trip cost`.

**PILLAR 2 — `/big-game-fishing-maldives`** — коммерческий хаб офшора. Ловит `big game / deep sea / sport fishing`, `luxury/private yacht fishing`.
Spokes: `gt/giant trevally`, `marlin`, `sailfish`, `popping jigging`, `trolling`, `yellowfin/dogtooth tuna`, `wahoo/mahi mahi`.

**PILLAR 3 — `/fly-fishing-maldives`** — нишевый авторитет (отстройка от Aardvark/COMO).
Spokes: `flats fishing`, `bonefish`, `permit/triggerfish`, `saltwater fly`.

**PILLAR 4 — `/maldives-fishing-packages`** ⭐ ГЛАВНАЯ КОНВЕРСИОННАЯ — ядро УТП «рыбалка + отель + менеджер». Ловит `fishing package/resort/holiday/vacation/liveaboard`.
Spokes («Who it's for»): `sunset fishing` (вход для пар), `family fishing`, `fishing honeymoon/couples`, + форматы (liveaboard / resort-stay / private-yacht).

**Перелинковка:** pillar↔spoke двусторонне; spoke↔spoke внутри кластера; **ВСЕ инфо- и видовые spokes ведут на Pillar 4** через CTA «book as a fishing-and-resort package» (инфо-трафик стекается в деньги). Сквозной CTA «Talk to a fishing concierge» → лид-форма.

### Quick wins (быстрый потенциал, слабая конкуренция)
`maldives fishing trip cost` · `family fishing maldives` · `maldives fishing honeymoon / couples` · `popping jigging maldives` · `bonefish maldives / flats fishing` · `dogtooth tuna / yellowfin / wahoo maldives` · `best fishing atolls / north male / ari atoll` · `maldives fishing license / rules` · `luxury / private yacht fishing charter maldives` · `sunset fishing maldives`.

### Как обойти FishingBooker и блоги
1. Главный ресурс — на **Pillar 4 (packages) + luxury/yacht** ключи: там нет ни одного fishing-first бренда.
2. Перебить «Complete Guide» **глубиной и свежестью** (помесячный календарь по видам, гид по атоллам, мост на пакет в каждом разделе).
3. **E-E-A-T как оружие:** именованные капитаны/гиды, фото трофеев с датами, видео, catch-and-release/правила — чего агрегатор показать не может.
4. **Сегментные посадочные** (honeymoon/family/couples + fishing) — пустое пересечение «рыбалка ∩ романтика/семья».
5. **Long-tail видов/техник/атоллов** пачкой — дешёвый авторитет домена → вес на pillars.

## Часть 2. RU + международная выдача

### RU-кластеры
RU-рынок узкий, но конкуренция слабая (выдачу держат агрегаторы/блоги без экспертизы; Яндекс силён на поведенческих → fishing-first бренд с фото трофеев/видео/именами гидов обходит их). Приоритет страниц:
1. **`рыбалка с проживанием / отель + рыбалка`** + **`рыболовный тур на мальдивы`** (ядро УТП, transactional).
2. **`троллинг/big game/на марлина`** + **`поппинг/джиггинг/GT`** (высокий чек, узкая экспертиза — главный RU-дифференциатор).
3. **`рыбалка на мальдивах`** (pillar-гайд, трафик-магнит) + `цена/отзывы/сезон`.
Под Яндекс: Яндекс.Бизнес, Вебмастер, отзывы, быстрые ответы.

### Матрица рынков (3 типа спроса)
- **Тип A — «fishing-travel»** (ищут рыболовный тур): **UK, US/CA/AU, Германия, Франция, Италия**, нишево Норвегия/Benelux/Япония, Индия (через EN). → играем органикой + экспертным контентом.
- **Тип B — «luxury-leisure»** (Мальдивы = медовый/премиум-отдых, рыбалка — апсейл «на месте»): **Залив (ОАЭ/Саудовская Аравия), Китай**. → заходим через luxury/honeymoon-контент и B2B-турагентов, НЕ через fishing-SEO.
- **Тип C — DIY/локальный:** не наша SEO-цель.

### Рекомендация по языкам на старте: **EN + DE + RU**
- **EN** (x-default) — покрывает ~70–80% платёжеспособного fishing-спроса (UK/US/CA/AU/Индия/Скандинавия/Benelux/частично Залив).
- **DE** — лучший не-EN ROI: самый зрелый рынок Angelreisen, высокий чек, немцы не охотно гуглят по-английски.
- **RU** — слабая конкуренция + домашний рынок + дешёвая локализация = «лёгкие деньги».
- Очередь: FR+IT (фаза 2) → ES → JA (фаза 3) → ZH/AR как отдельные не-SEO проекты.
- **Не распыляться** на 8 языков сразу (риск недолокализованных версий, конкурирующих за один EN-трафик).

## Часть 3. hreflang / международная архитектура
- **URL — subfolder на одном домене** (`/en/`, `/de/`, `/ru/`) — консолидирует вес на молодом домене, дешевле поддержки.
- **Полные reciprocal hreflang-кластеры** на каждой странице + обязательный **`x-default` → `/en/`**. Одна ошибка взаимности → Google игнорирует весь кластер (~75% внедрений с ошибками — это и есть дыра, которую закрываем).
- На старте — **только язык** (`en`,`de`,`ru`) без региона; региональные (`en-gb`) — только при разном контенте/ценах.
- **Самоссылающийся каноникал** на каждом языке (НЕ канонизировать перевод на EN — частая фатальная ошибка).
- **Локализация = транскреация**, не машинный перевод (термины: DE `Hochseeangeln/GT/Popping`, RU `троллинг/поппинг/джиггинг`, FR `pêche au gros`, IT `pesca d'altura`). Локализовать цены/кейсы/слаги/единицы/сезонность.
- **Чек-лист запуска:** reciprocal hreflang + x-default · самоканоникалы · языковой селектор без принудительного гео-редиректа по IP (Google краулит из США) · отдельные sitemap с hreflang в GSC + **Яндекс.Вебмастер** · регулярный аудит (Screaming Frog/Ahrefs).

## Часть 4. GEO / AI-поиск (ChatGPT, Perplexity, Google AI Overviews)

### Что AI цитирует сейчас
Доминируют **FishingBooker** (де-факто источник №1 для AI), Tom's Catch, Tripadvisor; нишевые fly/GT-операторы; блоги; и — важно — **курорты-конкуренты** (Park Hyatt `experienceparkhyattmaldives.com`, COMO Maalifushi) уже застолбили формат «resort experiences page». Wikipedia покрывает только промысловый тунец, не спорт-рыбалку. **Авторитетного бренд-источника по спорт-рыбалке нет** → свободно.

### Платформы (данные 2026)
ChatGPT ~48% топ-цитат из **Wikipedia**; Perplexity ~47% из **Reddit**; Google AIO ~23% из **YouTube** (и только Google показывает таймкоды). **YouTube ↔ AI-цитирование: корреляция 0.737 (сильнейший сигнал)**; просмотры/лайки почти не коррелируют (AI смотрит структуру/референсность, не популярность). 94% цитат — длинные видео, не Shorts.

### План citability (как писать под цитирование)
- **Вопросные H2/H3 = реальные запросы** («Is there fishing in the Maldives?», «Do you need a fishing licence?», «Best season for GT?», «How much does it cost?»).
- **Прямой ответ в первых 40–60 словах** секции (inverted pyramid).
- **Самодостаточные пассажи 134–167 слов** (AI выдёргивает фрагмент — он должен быть осмысленным сам по себе).
- Конкретные **факты+цифры+источник**; definition-блок в начале; **FAQ-блок 8–12 Q&A**; таблицы сравнения (сезон×вид×техника×атолл).
- Дата обновления + имя/био гида (E-E-A-T).

### Brand-сигналы (использовать активы клиента)
- **YouTube — главный рычаг:** бренд+ключ в названии и в первых 30 сек (транскрипт), длинные видео, **главы/таймкоды** с осмысленными названиями («Best season», «Licence & rules», «32 kg GT»), полные транскрипты/описания, линк видео↔страница сайта. EN-канал под EN-запросы, RU-канал под RU/Яндекс.
- **Wikipedia/Wikidata:** нейтрально расширить раздел рекреационной рыбалки (со ссылками на MSC/Минтуризма), завести Wikidata-сущность бренда, связать с каналом/сайтом.
- **Reddit/форумы** (ключ к Perplexity): экспертные ответы в r/saltwaterfishing, r/maldives, Tripadvisor Maldives, GT-popping тредах — органичные brand-mentions (×3 сильнее бэклинков).
- **Schema (JSON-LD):** `FAQPage`, `Article`(author/dates), `VideoObject`(transcript/chapters), `TouristTrip`/`Offer`, `LodgingBusiness`, `BreadcrumbList`.
- **`/llms.txt`** — markdown-карта citable-страниц (дёшево, на будущее).
- **robots.txt — допуск search-ботов** (`OAI-SearchBot`, `PerplexityBot`, `Claude-Web/Claude-SearchBot`, `Google-Extended`); тренировочные (`GPTBot`,`CCBot`) — по желанию. Блокировка search-ботов = вас нечем цитировать (частая фатальная ошибка).

### 7 GEO-ходов (impact / усилие)
1. **[оч.высокий/низкое]** Перепаковать YouTube под AI-извлечение (названия, главы, транскрипты, длинные видео).
2. **[оч.высокий/низкое]** Аудит robots.txt — открыть AI-search-ботов (за час).
3. **[высокий/среднее]** Pillar-гайд с answer-first структурой (перебить FishingBooker «first-hand» преимуществом).
4. **[высокий/среднее]** Полный JSON-LD комплект.
5. **[высокий/среднее]** Reddit/форумы под Perplexity.
6. **[сред-высокий/выше]** Wikipedia/Wikidata entity-работа.
7. **[средний/низкое]** `/llms.txt`.
**Метрика:** доля запросов (из 6+), где бренд/сайт/YouTube попадает в цитаты ChatGPT/Perplexity/AIO; share-of-voice vs Park Hyatt/COMO/FishingBooker — замер ежемесячно.

## Методология (для команды)
- **Кластеризация (SERP-overlap):** seed → 30–50 вариантов → классификация интента → попарное сравнение топ-10 (7–10 общих URL = одна статья, 4–6 = один кластер, 2–3 = перелинковка, 0–1 = раздельно) → hub-and-spoke (pillar 2500–4000 слов, spoke 1200–1800) → матрица перелинковки (нет страниц-сирот).
- **GEO Health (5 измерений):** Citability 25% · Structural Readability 20% · Multi-Modal 15% · Authority/Brand 20% · Technical Accessibility 20%.
- **Инструменты для боевого режима:** Google Search Console + Яндекс.Вебмастер; Keyword Planner/Wordstat (точные частоты); Ahrefs/Screaming Frog (hreflang-аудит); опц. DataForSEO MCP (живые данные). Готовый скил-комплект — [claude-seo](https://github.com/AgricIDaniel/claude-seo) (18 SEO-субагентов), методологию которого мы здесь и применили.
