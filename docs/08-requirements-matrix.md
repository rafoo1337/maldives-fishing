# 08 — Матрица требований (что хочет клиент)

> Источник — транскрипт созвона. Каждое требование привязано к тому, что реально сказал клиент.
> Категории: **Must** — явно нужно · **Should** — желательно/подразумевается · **Later** — фаза 2+/Модель 2 · **Out** — явно НЕ делаем (решение клиента/наше).

## Функциональные требования

| Требование (что сказал клиент) | Кат. | Наша трактовка / реализация |
|---|---|---|
| Мультиязычный сайт, **English первым** | **Must** | EN + 2–3 локали на старте (см. [14](14-seo-strategy.md)), остальное per-market в ретейнере |
| Рыбалка — герой, отель — крючок (fishing-first) | **Must** | ИА и воронка от рыбалки → отель ([12](12-information-architecture.md)) |
| Донести: на Мальдивах есть рыбалка мирового уровня (90% не знают) | **Must** | Hero + позиционирование ([11](11-positioning-messaging.md)) |
| 3 вида рыбалки (Fly / Big game / Sport) с описаниями | **Must** | 3 отдельные страницы + фактура ([13](13-content-strategy.md)) |
| Обучающие видео: основы рыбалки, **правила безопасности на борту** | **Must** | Раздел /safety + видео (повышает доверие, видео > бумажки) |
| FAQ (вопрос-ответ) | **Must** | /faq (+ FAQ-схема для AI-поиска) |
| Дефицит/эксклюзив: 6 отелей/~600 вилл, мало катеров, расписано на месяцы, предоплата 50%, штрафы | **Must** | Scarcity-блоки, календарь сезона, «осталось N окон» |
| Сайт **продаёт**: закрывает возражения, вызывает желание | **Must** | Конверсионная воронка, не визитка |
| **Лид → менеджер**, без онлайн-оплаты | **Must** | Многошаговая заявка → WhatsApp/менеджер |
| Квалифицированный лид (фильтрация: вид рыбалки, даты, кол-во, катер) | **Must** | Поля формы: направление → даты → Anglers+Leisure → контакты |
| CRM — копить лиды, не терять заявки (у конкурента уже есть оффер!) | **Must — в Фазу 1** | Форма → CRM (Битрикс/др.) + WhatsApp с 1-го дня; см. [15](15-tech-stack.md)/[18](18-roadmap-phases.md) |
| 2 номера (мальдивский + российский) + WhatsApp; по языкам — разные номера | **Must** | Контакты, гео/язык-зависимые |
| Корпоратив / ретриты — отдельная серьёзная страница | **Must** | /corporate-retreats (сегмент с высоким чеком) |
| Пары / семьи (муж рыбачит, семья рядом) | **Must** | Пакет «Fisher + Partner» + сегмент-страница |
| «Стань автором своего тура» (кастом) | **Should** | Путь кастом-заявки в воронке |
| Переиспользовать 40+ обзоров отелей + продакшн Влада | **Should** | Контент-хаб, видео-библиотека ([13](13-content-strategy.md)) |
| SEO + контекстная реклама по регионам/языкам | **Must** (ретейнер) | См. [14](14-seo-strategy.md) |
| Конфигуратор «выбор отеля + рыбалки → цена» / «как на booking» | **Should** | **Лёгкий конфигуратор-как-презентация** (не букинг): вид→отель→вилка «от $» → лид. **Лёгкая версия — в тариф B**; сложная логика/наличие — C/Модель 2. См. [09](09-inconsistencies.md) |
| Способы оплаты под разные регионы | **Later** | Не на сайте сейчас; через менеджера. См. [16](16-payments-legal.md) |
| Мобильное приложение | **Later** (Модель 2) | Задел в архитектуре бэкенда ([15](15-tech-stack.md)) |
| Свой флот / эксклюзивные контракты с отелями | **Later** (Модель 2) | Сайт проектируем, чтобы не мешал ([11](11-positioning-messaging.md)) |

## Явно НЕ делаем (Out)

| Не делаем | Почему |
|---|---|
| Личный кабинет пользователя | Решение клиента — не нужен |
| Онлайн-букинг отелей с выбором дат (как booking / выгрузка SAMO) | Требует лицензию SAMO (~$10k), РФ-юр. сложности, операционно тяжело; не нужно для Модели 1. См. [09](09-inconsistencies.md), [16](16-payments-legal.md) |
| Платёжный шлюз/онлайн-оплата на сайте сейчас | Чек высокий, продукт авторский, оплата серая (Дубай/нал/крипта) → менеджер. См. [16](16-payments-legal.md) |

## Нефункциональные / качественные
- **Премиальность** уровня Blue Safari/Alphonse — «мировой бренд не на шаблоне».
- **Скорость на мобайле** (дыра конкурента Boombastic — тяжёлые страницы).
- **Доверие**: видео-трофеи, капитаны-личности, отзывы с именами, прозрачные сезоны/цены «от».
- **Готовность к AI-поиску** (GEO) и корректный `hreflang` — см. [14](14-seo-strategy.md).
