## Заголовок
**Сроки — прозрачно**

| Этап | Срок | Вехи |
|---|---|---|
| **Discovery** | 1–2 недели | 1–2 созвона → документ стратегии + точная смета Фазы 1 |
| **Флагман — дизайн** | ~2–3 недели | Прототип → дизайн ключевых страниц → утверждение |
| **Флагман — разработка** | ~3–4 недели | Вёрстка, CMS, мультиязык, интеграции (CRM/WhatsApp) |
| **Флагман — контент + QA** | ~1–2 недели | Наполнение, тексты/видео, тестирование, перенос |
| **Запуск** | — | Домены, аналитика, обучение команды |
| **Ретейнер** | помесячно | Запускаем параллельно/после: SEO/AI, реклама, контент |

**Итого до запуска флагмана: ~6–10 недель** от старта (зависит от числа языков и готовности вашего контента — поэтому контент-пак важно зафиксировать в Discovery).
