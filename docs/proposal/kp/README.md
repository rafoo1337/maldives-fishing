## Как пользоваться
1. Открываешь главу по порядку → в ней готовый текст + `<!-- пометки по визуалу -->`.
2. Берёшь [раскладку 03](../03-deck-layout.md) — какой блок на какой странице/слайде.
3. Стиль — по [04](../04-visual-brand-direction.md); скриншоты конкурентов — по [shot-list](../../../assets/competitors/README.md).
4. Собираешь в Figma → PDF.
