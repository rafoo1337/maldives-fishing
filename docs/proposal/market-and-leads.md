> ⚠️ **Важно про «базы клиентов».** Чужие клиентские базы (персональные данные людей, оставленные НЕ нам) брать/выгружать нельзя — это нарушение GDPR и 152-ФЗ (те самые штрафы и блокировки, которых опасается клиент) и репутационный риск для премиум-бренда. ## 🎯 Как легально собрать клиентскую базу (плейбук)
Берём не чужие данные, а **доступ к аудитории** + растим **свою** базу с согласием:

1. **Листинг + реклама на маркетплейсах** (FishingBooker, Tom's Catch, GetMyBoat, OTA Viator/Klook) — приходишь к их аудитории официально, с брони идут реальные клиенты.
2. **Affiliate/реселл с премиум-операторами** (Yellow Dog, Frontiers, Aardvark, Farlows, Tailwaters, Sportquest) — они продают ваш продукт своей базе за 10–20%. **Это и есть легальный аналог «взять их клиентов».**
3. **Кросс-реферал** с операторами смежных направлений/сезонов (Сейшелы/Исландия/Аргентина) — обмен наводками за комиссию.
4. **Партнёрство с отелями/курортами** (вы уже так делаете — отели дают клиентов за %): усилить, добавить медиа-ролики в барах отелей под их гостей.
5. **Инфлюенсеры + YouTube** (свой канал + интеграции у BlacktipH/luxury-travel) → трафик на сайт → подписка/заявка.
6. **Таргет-реклама** в FB-группе «GT Popping Jigging», Reddit Ads (r/saltwaterfishing), Google по запросам — по интересам/гео, без чужих данных.
7. **Luxury-travel посредники** (Virtuoso preferred supplier, яхт-брокеры) — выход на HNW через тех, у кого уже есть доверие клиента.
8. **Свой сбор базы с согласием:** SEO/перехват трафика → лид-форма с opt-in; **лид-магнит** (PDF «сезоны и виды рыбалки на Мальдивах», чек-лист) в обмен на email; email/CRM-рассылка по своей базе. Это растёт законно и принадлежит вам.
9. **B2B-турагенты** (Scott Dunn-тип) — пакет «рыбалка + люкс-отель» для их состоятельных клиентов.

> Итог: к чужой аудитории — через рекламу/партнёрку/контент; свою базу — наращиваем с согласием (lead-magnet + opt-in). Это даёт тот же приток клиентов без 152-ФЗ/GDPR-рисков.
