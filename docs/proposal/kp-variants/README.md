# КП — 5 вариантов структуры (на выбор)

> Собрано 5 агентами параллельно: каждый — своя логика КП, детально по разделам (цель · что внутри · что продаёт · объём · сильные/слабые стороны). Открой любой файл, сравни. Внизу — рекомендация и гибрид.
> Общая рамка у всех: продаём **сайт-герой + мягкий апселл роста**, тон **50/50**, лаконичный поток + приложения.

## 🎯 Моя рекомендация
Михаил — **визионер-владелец**, который «смотрит и берёт не глядя на цену», но при этом считает маржу и называет конкурентов «пародией». Под него лучше всего ложится **гибрид**, а не один чистый вариант:

**Рекомендую каркас = Вариант 1 (Боль→Решение)** как самый логичный и сбалансированный, усиленный:
- **открытие/обложка — из Варианта 5** (эмоциональный крючок «Мальдивы, которые вам не продавали»);
- **блок рынка — из Варианта 2** (срочность «место №1 свободно, окно закроется» — бьёт в его дефицит-мышление);
- **цены/закрытие — из Варианта 4** (окупаемость через маржу, прагматично).

Получается дуга: **эмоция-крючок → боль и упущенная выгода → свободная ниша (срочность) → ваш бренд → сайт-герой → мягкий апселл роста → ROI и цены → Discovery.**
